🔥 FAF Web API Template

Production-ready шаблон для создания ASP.NET Core 10 микросервисов.

Быстрый старт

dotnet new install Faf.Templates.WebApi
dotnet new faf-webapi -n MyService
cd MyService
dotnet run

Что включено

Компонент Описание
Clean Architecture Controllers → Services → Repositories
Result Pattern Типобезопасная обработка ошибок
Serilog Структурированное логирование
OpenTelemetry Трейсинг, метрики, логи
JWT Auth Bearer-токены, policy-based авторизация
Health Checks Liveness + Readiness на порту 8081
202 теста Unit + Integration через TUnit
